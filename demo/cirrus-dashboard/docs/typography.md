# Typography

A six-step type scale. All sizes scale with the user's preferred font size (no fixed pixel headings).

## Type scale

| Token | Size | Line height | Use |
|---|---|---|---|
| `text-xs` | 0.75rem | 1rem | Captions, axis labels |
| `text-sm` | 0.875rem | 1.25rem | Secondary body, table cells |
| `text-base` | 1rem | 1.5rem | Body text |
| `text-lg` | 1.125rem | 1.75rem | Lead paragraphs |
| `text-xl` | 1.25rem | 1.75rem | Card titles |
| `text-2xl` | 1.5rem | 2rem | Section headings |
| `text-3xl` | 1.875rem | 2.25rem | Page titles |

## Font stacks

| Stack | Family |
|---|---|
| Sans (body, headings) | `Inter, system-ui, sans-serif` |
| Mono (code, station IDs) | `JetBrains Mono, ui-monospace, monospace` |
| Tabular (numbers in tables) | `Inter, system-ui, sans-serif` with `font-variant-numeric: tabular-nums` |

## Weight scale

| Token | Weight | Use |
|---|---|---|
| `font-normal` | 400 | Body text |
| `font-medium` | 500 | Table headers, emphasized inline |
| `font-semibold` | 600 | Card titles, section headings |
| `font-bold` | 700 | Page titles |
